Soft status pill for player availability, match state, and the "AI" marker.

```jsx
<Tag tone="green">M. Chen</Tag>
<Tag tone="amber">Doubtful</Tag>
<Tag tone="red">In progress</Tag>
<Tag tone="inverse" icon="cpu">AI</Tag>
```

Tones: `green` (available / on-plan), `amber` (caution / doubtful), `red` (risk / live), `neutral`, `inverse` (dark, for the AI badge).
