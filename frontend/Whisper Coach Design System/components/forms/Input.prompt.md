Labelled text input or textarea, with the brand green focus ring.

```jsx
<Input label="Opponent" placeholder="e.g. FC Riverside" />
<Input label="Notes (optional)" multiline rows={2}
       placeholder="Their striker is fast, watch the flanks" />
```

Pass `multiline` for a textarea. Omit `label` to render a bare control. All native input/textarea attributes pass through.
