Developer reference

Single-page spec for building the Whisper Coach Flutter app. Every value below is a CSS custom property from styles.css — map them into a Flutter theme / token file. Components are rendered live from the design-system bundle so you can read their real states. Brand: emerald #1D9E75, pitch greens, warm-stone neutrals; flat, hairline-bordered, iOS-native.

Flutter mapping: token --green-500 → const Color(0xFF1D9E75); radii → BorderRadius.circular(); the 0.5px hairline → a Border.all(width:0.5, color: Color(0x1F000000)). Icons: use the tabler_icons package or bundle the SVGs you need via flutter_svg.

Colors

One interactive accent (emerald), grass pitch greens, a warm-stone neutral ramp, and soft tactical status colors. No gradients on chrome.

Brand green — interactive / accent

Typography

System sans carries the UI (weights 400/500). Hanken Grotesk is the display face for the wordmark, scores and headlines. Tabular numerals for stats & match minutes.

Spacing, radii & elevation

4px base grid. Cards are flat with a hairline border; shadow is reserved for floating layers.

Motion

Quiet & quick. 150ms standard transitions; presses scale ~0.96; no decorative looping.
