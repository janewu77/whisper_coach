## 🚀 Hackathon MVP (Minimal Version)

Goal:
👉 A "fast match management + AI on-the-fly lineup + post-match summary" tool

---

## 🧠 Core Flow (must keep)

### 1. Quick team setup (ultra-lightweight input)

* 📸 Take a photo / upload the team roster (player list image)
* AI (PydanticAI) → structure it into a players list

---

### 2. Match creation (minimal input)

* 📱 Screenshot / enter opponent info
* 📍 Venue info (manual or OCR)

Structured:

```json
{
  "opponent": "...",
  "location": "...",
  "date": "...",
  "notes": "..."
}
```

---

### 3. Automatic formation arrangement (the core highlight ⭐)

Input:

* Player list
* Opponent info
* Optional: match type (strong/weak)

Output:

* 4-3-3 / 4-2-3-1 / 3-5-2

```json
{
  "formation": "4-3-3",
  "lineup": [
    {"player": "John", "position": "ST"},
    {"player": "David", "position": "CM"}
  ],
  "reason": "Opponent strong midfield, recommend compact center"
}
```

---

### 4. On-the-fly adjustments (during the match)

* Tap a player
* Enter voice / text:

  * "John is too tired"
  * "The left flank is getting overrun"

AI output:

* Substitution suggestions
* Position adjustments

👉 Key point: **no complex UI, just buttons + a simple pitch diagram**

---

### 5. Post-match summary (auto-generated by AI)

Input:

* Formation
* On-the-fly adjustments
* Voice notes

Output:

* Match summary
* Player performance
* Improvement suggestions

---
